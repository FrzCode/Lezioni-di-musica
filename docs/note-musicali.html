Le note musicali

Le note musicali del nostro sistema musicale sono sette: do, re, mi, fa, sol, la, si.

Sul pentagramma si scrivono sulle linee o sugli spazi. Il criterio per individuarle è far riferimento alla chiave e tenere conto di quest'ordine.

È opportuno imparare le note anche nell'ordine inverso: do, si, la, sol, fa, mi, re e per salti: do, mi, sol, si, re, fa, la; do, la, fa, re, si, sol, mi.
